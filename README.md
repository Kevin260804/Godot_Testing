# Godot_Testing
This repository is a personal learning project to help me get started with game development using the Godot Engine.

## Purpose
The goal of this project is to experiment with Godot features, learn the basics of game development, and build simple prototypes. It serves as a sandbox for testing ideas and understanding how game engines work.

## Notes
- This project is for educational purposes and may contain experimental or incomplete features.
- Contributions are not expected, but feedback is welcome.
